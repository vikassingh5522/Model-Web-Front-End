

## üéØ **CSS Units Explained**

---

### üîπ **1. Absolute Units**

> Fixed ‚Äî do **not** change based on screen size or parent elements.

| Unit | Meaning                  | Example            | Notes                            |
| ---- | ------------------------ | ------------------ | -------------------------------- |
| `px` | Pixels                   | `width: 200px;`    | Commonly used for fixed layouts. |
| `pt` | Points (1pt = 1/72 inch) | `font-size: 12pt;` | Used mostly for print media.     |
| `cm` | Centimeters              | `width: 5cm;`      | Physical unit.                   |
| `mm` | Millimeters              | `height: 10mm;`    | Physical unit.                   |
| `in` | Inches (1in = 96px)      | `width: 2in;`      | Physical unit, rarely used.      |

üß† **Tip:** Absolute units are rarely used for web ‚Äî more for print styles.

---

### üîπ **2. Relative Units**

> Scale **relative** to something (parent, root, or viewport).

| Unit   | Relative To              | Example              | Explanation                                        |
| ------ | ------------------------ | -------------------- | -------------------------------------------------- |
| `%`    | Parent element           | `width: 50%;`        | 50% of parent‚Äôs width.                             |
| `em`   | Parent‚Äôs font size       | `font-size: 2em;`    | If parent font-size = 20px ‚Üí 2em = 40px.           |
| `rem`  | Root (`html`) font size  | `font-size: 1.5rem;` | If root font-size = 16px ‚Üí 1.5rem = 24px.          |
| `vw`   | Viewport width           | `width: 50vw;`       | 50% of browser width.                              |
| `vh`   | Viewport height          | `height: 50vh;`      | 50% of browser height.                             |
| `vmax` | Larger of `vw` or `vh`   | `font-size: 10vmax;` | Depends on whichever is bigger (width or height).  |
| `vmin` | Smaller of `vw` or `vh`  | `font-size: 10vmin;` | Depends on whichever is smaller (width or height). |
| `ch`   | Width of ‚Äú0‚Äù character   | `width: 40ch;`       | Useful for text areas or monospace layouts.        |
| `ex`   | x-height of current font | `height: 2ex;`       | Rarely used; relative to lowercase ‚Äúx‚Äù height.     |


1 vw= 5px
50vw = 250px
---

### üß© **3. Examples**

```css
html {
  font-size: 16px;  /* 1rem = 16px */
}

.container {
  width: 80%;       /* 80% of parent width */
  font-size: 1.5em; /* 1.5 √ó parent's font-size */
}

.text {
  font-size: 2rem;  /* 2 √ó root font-size = 32px */
  margin: 5vh 10vw; /* responsive to viewport */
}
```

---

### üí° **Quick Recap**

| Type         | Unit Examples                                            | Depends On                |
| ------------ | -------------------------------------------------------- | ------------------------- |
| **Absolute** | `px`, `pt`, `cm`, `mm`, `in`                             | Fixed, no scaling         |
| **Relative** | `%`, `em`, `rem`, `vw`, `vh`, `vmax`, `vmin`, `ch`, `ex` | Parent, root, or viewport |
